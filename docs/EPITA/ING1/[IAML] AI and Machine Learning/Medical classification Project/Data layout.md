The dataset contains cases of two type of breast cancer:

- malignant (M)
- benign: (B)

along with the features of the certain case with an ID.

## Features

- Radius
- Texture
- Perimeter
- Area
- Smoothness
- Compactness
- Concavity
- Concave points
- Symmetry

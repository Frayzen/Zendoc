# Energy
On $\mathcal{L}^{2}(\R)$
$E_x=\langle x,x \rangle =||x||^2=\int_\R|x(t)|^2dt$
  
# Average Power
On $\mathcal{L}^{pm}(\R)$
$P_x=\underset{T\rightarrow \infty} {lim} \frac{1}{T} \int_{-T/2}^{T/2}|x(t)|^2dt = \langle x,x \rangle$
If the signal is T-periodic
$P_x = \frac{1}{T}\int_{-T/2}{T/2}|x(t)|^2dt$
